# openpath-first-pr

This repository helps the community take a first step into open-source contribution. It is a safe, welcoming place for beginners to experience the real GitHub workflow and build confidence before contributing to larger projects.

## Who this repo is for

- Students
- Beginners
- First-time contributors

## What you will learn

- Forking a repository
- Creating a commit
- Opening a pull request
- Understanding review feedback

## Simple rules

- One PR per person.
- No spam and no private information.
- Keep changes small and respectful.

## The single allowed task

Add your name to `participants.md`.

## Contribution flow

1. Fork this repo.
2. Clone your fork.
3. Edit `participants.md` and add one line.
4. Commit your change.
5. Open a pull request.

## Why this helps

This practice mirrors the core workflow used across open-source projects. After completing it, you will be ready to contribute to real repositories with confidence.
